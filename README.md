I read AI agent source code so you don't have to. I've gone through 15 production codebases line by line — Claude Code, Codex CLI, Dify, Goose, OpenHands, and others — and published architecture teardowns with diagrams, code evidence, and cross-project comparisons.

Background: a decade building large-scale distributed systems. Now focused on AI agent internals — how they manage context, handle security, orchestrate tools, and coordinate multiple models. I also build tools: a quantitative finance engine with genetic algorithm strategy evolution, a security framework for AI agents, and a behavioral testing framework.
